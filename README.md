# MiRemovePendrive

Remove pendrive safely.

## Information

 - [Page MiRemovePendrive](https://github.com/mugomes/miremovependrive)

## System Requirement

### Linux

 - Ubuntu 24.04 or higher
 - Graphics: GPU with OpenGL support (required for 3D acceleration)

### Windows

- Windows 10 or higher
- Graphics: GPU with OpenGL support (required for 3D acceleration)
